📄 **Geração de Onda com PIO no RP2040 (Pico W)**

📌 Descrição

Este projeto demonstra o uso do periférico PIO (Programmable I/O) do microcontrolador RP2040 para geração de uma onda digital (square wave) em um pino GPIO.

O PIO permite criar periféricos customizados em hardware, com alta precisão temporal e baixo uso da CPU.

🎯 Objetivos
- Compreender o funcionamento do PIO do RP2040
- Gerar sinais digitais utilizando máquina de estados (State Machine)
- Controlar frequência de saída com precisão
- Integrar código C com programas .pio

🧪 Sugestão de Experimentos
- Variar a frequência do sinal
- Medir com osciloscópio
- Comparar com PWM tradicional
- Implementar múltiplas saídas
